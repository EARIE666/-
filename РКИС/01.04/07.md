## Задание 7. Сравнительный анализ инструментов для графического интерфейса баз данных

| Инструмент | Поддерживаемые СУБД | Лицензия | Преимущества | Недостатки |
| :--- | :--- | :--- | :--- | :--- |
| DBeaver | MySQL, PostgreSQL, Oracle, SQL Server, SQLite, DB2 и более 30 других | Open Source (Community), Enterprise (платная) | Универсальность — работает с любыми СУБД<br>Кроссплатформенность (Windows, macOS, Linux)<br>Встроенный редактор ER-диаграмм<br>Поддержка Git для версионирования скриптов | Требователен к ресурсам<br>Интерфейс может быть перегружен для новичков |
| DataGrip | 30+ СУБД | Платная (JetBrains, есть триал) | Интеллектуальное автодополнение (контекстное)<br>Глубокий анализ и рефакторинг кода<br>Интеграция с Git и системами CI/CD | Высокая стоимость подписки<br>Требует времени на освоение |
| pgAdmin | Только PostgreSQL | Open Source | Официальный инструмент для PostgreSQL<br>Удобное веб-управление серверами<br>Встроенный мониторинг производительности | Работает только с PostgreSQL<br>Интерфейс считается устаревшим |
| MySQL Workbench | Только MySQL / MariaDB | Open Source | Официальный инструмент от Oracle<br>Мощный модуль моделирования баз данных<br>Инструменты миграции из других СУБД | Только MySQL<br>Нестабильность на некоторых версиях Windows |
| Microsoft SSMS | MS SQL Server | Бесплатный | Лучшее решение для SQL Server<br>Встроенные профилировщики и отладчики<br>Глубокая интеграция с экосистемой Microsoft | Только Windows<br>Только MS SQL Server |
| Navicat | MySQL, PostgreSQL, SQLite, MongoDB и др. | Платная | Элегантный и интуитивный интерфейс<br>Мощные инструменты импорта/экспорта<br>Удобная синхронизация данных между базами | Высокая стоимость лицензий<br>Нет бесплатной версии для коммерческого использования |
| Beekeeper Studio | MySQL, PostgreSQL, SQLite, SQL Server | Open Source (Community) | Простой и минималистичный интерфейс<br>Быстрый запуск<br>Хорошо подходит для начинающих | Ограниченный функционал по сравнению с DBeaver<br>Меньше возможностей администрирования |
